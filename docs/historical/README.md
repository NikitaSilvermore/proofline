# docs/historical/

Superseded docs kept for the record only. **Anything in this folder is
excluded from the Claude-project sync** (`scripts/sync-claude-project.mjs`
never copies `docs/historical/*`) — a superseded doc uploaded to the project
is a doc an agent reads *without* its banner, which silently poisons chats.

If a doc here still matters, it has been replaced by a current doc elsewhere;
follow that one. Empty for now.
